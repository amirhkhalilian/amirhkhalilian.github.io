Introduction

Non-negative matrix factorization is a powerful tool for dimensionality reduction and data analysis. The goal in matrix factorization, in general, is to summarize a given data matrix with two low-rank factors. In this non-negative matrix factorization (NMF), we require the factor matrices to have only non-negative elements. This is in contrast to some other factorization methods, such as principle component analysis (PCA). In this brief article, we are interested in scenarios where not all the elements in the given data matrix are measured or valid. In this case, we want to "skip" these missing elements when solving the optimization problem and find the factor matrices without considering the error on those elements.

The non-negative matrix factorization problem

Given the non-negative data matrix D∈R≥0m×nD\in\mathbb{R}^{m\times n}_{\geq 0}D∈R≥0m×n​ we want to find matrices W∈R≥0m×rW\in\mathbb{R}^{m\times r}_{\geq 0}W∈R≥0m×r​ and H∈R≥0r×nH\in\mathbb{R}^{r\times n}_{\geq 0}H∈R≥0r×n​ such that D≈WHD\approx WHD≈WH. The optimization problem can be written as

W∗,H∗=arg⁡min⁡W,H∥D−WH∥F2s.t.W∈Rm×r,W≥0 and H∈Rr×n,H≥0. W^{\ast}, H^{\ast} = \arg\min_{W,H} \|D-WH\|_{F}^2 \quad \text{s.t.} \quad W\in\mathbb{R}^{m\times r},\quad W\geq 0 \text{ and } H \in \mathbb{R}^{r\times n},\quad H\geq 0.W∗,H∗=argW,Hmin​∥D−WH∥F2​s.t.W∈Rm×r,W≥0 and H∈Rr×n,H≥0.

An iterative algorithm

The following algorithm, called Lee and Seung's multiplicative update rule, solves the NMF problem.

  • initialize: W and H with random non-negative elements

  • repeat until convergence:

H[i,j]n+1←H[i,j]n((Wn)TV)[i,j]((Wn)TWnHn)[i,j]{\displaystyle \mathbf {H} _{[i,j]}^{n+1}\leftarrow \mathbf {H} _{[i,j]}^{n}{\frac {((\mathbf {W} ^{n})^{T}\mathbf {V} )_{[i,j]}}{((\mathbf {W} ^{n})^{T}\mathbf {W} ^{n}\mathbf {H} ^{n})_{[i,j]}}}}H[i,j]n+1​←H[i,j]n​((Wn)TWnHn)[i,j]​((Wn)TV)[i,j]​​ W[i,j]n+1←W[i,j]n(V(Hn+1)T)[i,j](WnHn+1(Hn+1)T)[i,j].{\displaystyle \mathbf {W} _{[i,j]}^{n+1}\leftarrow \mathbf {W} _{[i,j]}^{n}{\frac {(\mathbf {V} (\mathbf {H} ^{n+1})^{T})_{[i,j]}}{(\mathbf {W} ^{n}\mathbf {H} ^{n+1}(\mathbf {H} ^{n+1})^{T})_{[i,j]}}}}.W[i,j]n+1​←W[i,j]n​(WnHn+1(Hn+1)T)[i,j]​(V(Hn+1)T)[i,j]​​.

One drawback of this algorithm is that when an entry of WWW or HHH is initialized as zero or positive, it remains zero or positive throughout the iterations. Therefore, all entries should be initialized to be positive and sparsity constraints are not simply added to the algorithm unless via hard-thresholding.

Alternating non-negative least squares algorithm

Another algorithmic solution can be achieved via alternating non-negative least squares (ANLS). Assume we have WWW fixed then we get the following optimization for HHH:

arg⁡min⁡H∥D−WH∥F2s.t.H∈Rr×n,H≥0,\arg\min_{H} \|D-WH\|_{F}^2 \quad \text{s.t.} \quad H \in \mathbb{R}^{r\times n},\quad H\geq 0,argHmin​∥D−WH∥F2​s.t.H∈Rr×n,H≥0,

which is the least squares problem for HHH with a non-negativity constraint. The solution to this problem can be determined by solving W⊤WH=W⊤DW^\top W H=W^\top DW⊤WH=W⊤D for HHH followed by a projection onto R≥0\mathbb{R}_{\geq 0}R≥0​. We can also use the method proposed in this paper to solve this problem. The code can be found here. Now for fixed HHH we can write:

arg⁡min⁡W∥D⊤−H⊤W⊤∥F2s.t.W∈Rm×r,W≥0,\arg\min_{W} \|D^\top-H^\top W^\top\|_{F}^2 \quad \text{s.t.} \quad W \in \mathbb{R}^{m\times r},\quad W\geq 0,argWmin​∥D⊤−H⊤W⊤∥F2​s.t.W∈Rm×r,W≥0,

This can be minimized by solving HH⊤W⊤=HD⊤H H^\top W^\top = H D^\topHH⊤W⊤=HD⊤ for WWW, followed by the projection onto R≥0\mathbb{R}_{\geq 0}R≥0​. It is also recommended that we normalize the columns of WWW to unit ℓ2\ell_2ℓ2​-norm and scale the rows of HHH accordingly after each iteration. Using changes in ∥D−WH∥F\|D-WH\|_F∥D−WH∥F​ per iteration, we can define the stopping criteria.

Masked non-negative matrix factorization

In real application, we might have a case where the columns of DDD represent different data-points and each row represents a separate features. In this case, we might have a situation where some of the features are missing from the dataset. In the NMF formulation, we do not want to consider the error over these missing values. In other words, we only wanna consider the error on the elements in DDD that are valid or measured and discard the error on the rest. It is fair to assume that we know which elements are missing and use this information when formulating the problem.

Let M∈{0,1}m×nM\in\{0,1\}^{m\times n}M∈{0,1}m×n be the masking matrix such that Mij=0M_{ij}=0Mij​=0 iff DijD_{ij}Dij​ is missing, otherwise Mij=1M_{ij}=1Mij​=1. In this case we only want to consider the error on measured signals. So we get,

W∗,H∗=arg⁡min⁡W,H∥M∘(D−WH)∥F2s.t.W∈Rm×r,W≥0 and H∈Rr×n,H≥0, W^{\ast}, H^{\ast} = \arg\min_{W,H} \|M\circ(D-WH)\|_{F}^2 \quad \text{s.t.} \quad W\in\mathbb{R}^{m\times r},\quad W\geq 0 \text{ and } H \in \mathbb{R}^{r\times n},\quad H\geq 0,W∗,H∗=argW,Hmin​∥M∘(D−WH)∥F2​s.t.W∈Rm×r,W≥0 and H∈Rr×n,H≥0,

where ∘\circ∘ denotes the element-wise multiplication. The loss function in this case only considers the error in elements of DDD where MMM is non-zero.

How do we add the Masking to an iterative solver?

We can apply the ANLS algorithm to the masked problem. When we fix WWW we get the following optimization over HHH,

arg⁡min⁡H∥M∘(D−WH)∥F2s.t.H∈Rr×n,H≥0.\arg\min_{H} \|M\circ(D-WH)\|_{F}^2 \quad \text{s.t.} \quad H \in \mathbb{R}^{r\times n},\quad H\geq 0.argHmin​∥M∘(D−WH)∥F2​s.t.H∈Rr×n,H≥0.

Following the ANLS method we have to solve W⊤M∘(WH)=W⊤(M∘D)W^\top M\circ(W H)=W^\top (M\circ D)W⊤M∘(WH)=W⊤(M∘D) for HHH followed by a projection onto R≥0\mathbb{R}_{\geq 0}R≥0​. The derivation here shows the solution without the non-negative constraint. Let M=[m1,⋯ ,mn]M=[m_1,\cdots,m_n]M=[m1​,⋯,mn​] where mjm_jmj​ is the jjj-th column of MMM. Then, these equations can be solved for each column of HHH separately by solving the linear system W⊤diag(mj)W hj=W⊤diag(mj) djW^\top \mathrm{diag}(m_j) W\,h_j=W^\top \mathrm{diag}(m_j)\,d_jW⊤diag(mj​)Whj​=W⊤diag(mj​)dj​ for hjh_jhj​. We note that this can still be solved by nnlsm_blockpivot.m method from this paper. When an element in mjm_jmj​ is zero, we remove the corresponding row from the linear system (technically there is no need to do this and we can directly form W⊤diag(mj)WW^\top \mathrm{diag}(m_j) WW⊤diag(mj​)W and W⊤diag(mj) djW^\top \mathrm{diag}(m_j)\,d_jW⊤diag(mj​)dj​ and solve the linear system, however, by removing the zeros we make sure that W⊤diag(mj)WW^\top \mathrm{diag}(m_j) WW⊤diag(mj​)W does not become unnecessarily ill-conditioned).

Now for fixed HHH we can write:

arg⁡min⁡W∥M⊤∘(D⊤−H⊤W⊤)∥F2s.t.W∈Rm×r,W≥0,\arg\min_{W} \|M^\top\circ(D^\top-H^\top W^\top)\|_{F}^2 \quad \text{s.t.} \quad W \in \mathbb{R}^{m\times r},\quad W\geq 0,argWmin​∥M⊤∘(D⊤−H⊤W⊤)∥F2​s.t.W∈Rm×r,W≥0,

which is solved by the system of equations H M⊤∘(H⊤W⊤)=HD⊤H\,M^\top \circ (H^\top W^\top) = H D^\topHM⊤∘(H⊤W⊤)=HD⊤ for WWW followed by a projection onto R≥0\mathbb{R}_{\geq 0}R≥0​. Similarly, this can be diagonalized for rows of the matrix WWW and solved via the same method.

A note on implementation in Matlab

I implemented this algorithm in Matlab and you can find the code here. The core solver is the masked-nnmf function. Here is a brief overview of the main iteration.

The solver function receives three main inputs:

  • D: the data matrix of size m x n.

  • M: the mask matrix of size m x n with elements Mij = 0 iff Dij is missing.

  • r: desired rank.

The main loop consists of an update over HHH followed by an update over WWW. Given the matrix M we can precompute the rows in WjW_jWj​ and djd_jdj​ that we need to use when solving for hjh_jhj​. Similarly, we find columns to use in each row of did_idi​ and HiH_iHi​ when solving for rows wiw_iwi​. We write the update iteration as:

% update H
WT = W';
parfor (j = 1:size(D,2), num_cores) % loop over columns of D
	WT_mj_dj = transpose(sum(W(rows2use{j},:).*D(rows2use{j},j),1));
	WT_mj_W = WT(:,rows2use{j}) * W(rows2use{j},:);
	H(:,j) = nnlsm_blockpivot(WT_mj_W, WT_mj_dj, true);
end

% update W
HT = H';
parfor (i = 1:size(D,1), num_cores) % loop over rows of D
	H_mi_di = sum(H(:,cols2use{i}).*D(i,cols2use{i}),2);
	H_mi_H = H(:,cols2use{i}) * HT(cols2use{i},:);
	W(i,:) = nnlsm_blockpivot(H_mi_H, H_mi_di, true);
end
