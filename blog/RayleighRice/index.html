Problem in hand

Consider the data with all positive values denoted as sns_nsn​ for all n∈{1,⋯ ,N}n\in\{1,\cdots,N\}n∈{1,⋯,N}. It is desired to find an unsupervised threshold τs\tau_sτs​ such that we say for sn>τss_n>\tau_ssn​>τs​ the data point nnn belongs to a class otherwise the data point belongs to the other class. In order to find such a threshold without introducing any extra parameters we use a Rayleigh-Rice mixture model. Note that the values of sns_nsn​ are positive and Rayleigh-Rice mixture models these values very well. Once the Rayleigh-Rice mixture model is fitted, the intersection of the two distributions is the threshold value.
